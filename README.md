## 题目描述
输入两个整数 $a, b$，输出它们的和（$|a|,|b| \le {10}^9$）。
**注意**
1. 有负数哦！
2. C/C++ 的 main 函数必须是 `int` 类型。程序正常结束时的返回值必须是 0。
## 输入格式
两个以空格分开的整数。
## 输出格式
一个整数。
