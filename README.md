# git-学习报告
## 一、常用指令学习
|指令|作用|
|---|---|
|git clone|从git服务器拉取代码|
|git config|配置开发者用户名和邮箱|
|git branch|创建、重命名、查看、删除项目分支|
|git checkout|切换分支|
|git status|查看文件变动状态|
|git add|添加文件变动到暂存区|
|git commit|提交文件变动到版本库|
|git push|将本地的代码改动推送到服务器|
|git pull|将服务器上的最新代码拉取到本地|
|git log|查看版本提交记录|
|git tag|为项目标记里程碑|
