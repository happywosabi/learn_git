# Git 实验报告 (zhc)

## 1. Git 常用指令学习：
- `git clone`: 克隆代码库到本地
- `git checkout -b`: 创建新分支
- `git add`: 添加文件
- `git commit`: 提交更改
- `git push`: 上传代码
- `git pull`: 拉取更新
- `git status` 检查状态
